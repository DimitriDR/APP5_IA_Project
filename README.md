# Installation
## Requirements
- Python 3.12 (may work with older versions)
- pip

## Install dependencies with requirements.txt
```bash
pip install -r requirements.txt
```

It is possible that punkt is not installed, if so, run:
```bash
python
nltk.download('punkt')
```
